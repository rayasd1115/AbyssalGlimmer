# Abyssal Glimmer 遊戲指南

歡迎來到《深淵微光》！這是一份幫助你深入深淵、挑戰終極魔王的完整攻略。

## 遊戲目標

你的目標是成功闖過5層深淵，到達第6層的最終關卡，並擊敗在那裡等待你的強大魔王以贏得遊戲。

## 基本操作

- **W/A/S/D**: 移動你的角色（@）。你的角色會朝著移動的方向。
- **E**: 進行一次近戰攻擊。
- **F**: 朝你面對的方向發射遠程攻擊。
- **ESC**: 退出遊戲。

## 遊戲元素詳解

### 介面（HUD）與玩家初始數值

顯示在螢幕最上方，包含了玩家的即時狀態與基本數值。

- **HP (生命值)**: 你的健康狀況。**初始值為100**，當降至0時遊戲結束。
- **SHIELD (護盾)**: 在生命值之前吸收傷害。會隨著時間緩慢回復（大約每0.6秒回復3點）。**初始值為0**。
- **LEVEL (層數)**: 你目前所在的深淵層數。
- **GOLD (金幣)**: 用於購買道具和升級的貨幣，上限為1000。**初始為0**。
- **MELEE (近戰)**: 你的近戰攻擊傷害。**初始為8**。
- **RANGED (遠程)**: 你的遠程攻擊傷害。**初始為8**。

### 道具

走過即可拾取。

- **+ (藥水)**: 回復10點生命值。
- **\* (金幣)**: 獲得5枚金幣。

### 一般怪物 (Z)

- **出現**: 在你移動幾步後，怪物便會出現。
- **行為**: 當你進入牠們的視野範圍（8格）時，牠們會追擊你。
- **屬性**:
    - **基礎生命值 (HP)**: 20 (每深入一層 +2)
    - **基礎傷害 (Damage)**: 10 (每深入一層 +1.5)
- **獎勵**: 擊敗一隻敵人可獲得5枚金幣。

### 出口 (>)

找到並走上去即可前往下一層。

## 遊戲進程與攻略

### 樓層 1-4：探索與戰鬥

- 在這些樓層，你需要與怪物戰鬥，收集金幣和藥水，並找到通往下一層的出口。
- 在找到出口後，你可以選擇花費金幣來升級你的遠程攻擊力（+2傷害）。初始費用為20金幣，每次升級後費用會增加5金幣。

### 樓層 5：商人營地

- 這是一個**安全區域**，沒有任何怪物。
- 你會在這裡找到一位商人（M），可以與他進行交易。
- **商店貨品**:
    - **藥水 (10金幣)**: 回復10點生命值。
    - **完全治療 (50金幣)**: 將生命值完全回復到100。
    - **遠程升級 (初始20金幣)**: 永久提升2點遠程攻擊力。
    - **遠程藥劑 (30金幣)**: 將你的遠程傷害和射程提升至10（如果當前值較低）。
    - **近戰藥劑 (20金幣)**: 將你的近戰傷害提升至20（如果當前值較低）。
    - **護盾藥劑 (60金幣)**: 永久增加50點護盾上限，並回復50點護盾。

### 樓層 6：魔王巢穴

這是最終的決戰之地！這裡只有唯一的敵人：最終魔王。

#### 最終魔王 (Boss)

- **總生命值 (HP)**: 200
- **攻擊傷害**: 20 (無論是能量球還是身體碰撞)

---

#### **階段一：固定炮台 (HP 200 -> 101)**

- **行為**: 在此階段（消耗BOSS前100點血量），魔王是固定的，無法移動。它會朝八個方向輪流發射能量球（`O`）。
- **攻略**: 保持移動，利用攻擊間隙進行遠程或近戰輸出。這是相對安全的輸出階段。

---

#### **轉變階段 (HP降至100時觸發)**

- **行為**: 當生命值低於100時，魔王會進入轉變階段。此階段它會閃爍並伸長身體，為下一階段做準備。
- **攻略**: 在此階段，魔王是**無敵的**。你需要做的就是躲避，等待它完成變形。

---

#### **階段二：貪食蛇 (HP 100 -> 0)**

- **行為**: 轉變完成後，魔王變形為一條長蛇（`Bosssssss...`），並會主動追擊你。此階段將持續到魔王被擊敗。
- **弱點**: 只有蛇頭（`B`）可以被攻擊並受到傷害。攻擊身體其他部位是無效的。
- **攻略**:
    - 這是最危險的階段。你需要非常靈活地走位，引誘蛇頭攻擊，然後在躲避後迅速反擊。
    - **遠程攻擊**是此階段最安全的輸出方式。
    - 盡量與蛇身保持距離，避免被其包圍或觸碰。利用場地中的障礙物來卡住它的移動路線，為自己創造攻擊機會。

---

擊敗魔王後，恭喜你！你已經成功征服了深淵！
